Microsoft Power Platform · Enablement Reference

Power Platform Admin — Starter Kit

A curated path of free, public Microsoft resources for someone standing up the Power Platform admin function from scratch. Sequenced from day-one foundations to running a governed platform at scale.

1

Start here — foundations

Build the mental model. If you come from a Power BI background, much of this will feel familiar.

→Power Platform Fundamentals (PL-900)

The ideal on-ramp — a free, guided learning path plus an entry-level certification covering the whole platform (Power Apps, Power Automate, Power Pages, Copilot Studio, Dataverse).

learn.microsoft.com/en-us/credentials/certifications/power-platform-fundamentals/

→Administrator training catalog

Every free Microsoft Learn module filtered to the administrator role — self-paced, hands-on, and searchable by topic.

learn.microsoft.com/en-us/training/browse/?products=power-platform&roles=administrator
2

The admin's home base

The console you'll live in, and the security model underneath it.

→Power Platform Admin Center — overview

Your day-to-day cockpit: environments, data policies (DLP), capacity, and analytics. Start here to learn what each area does.

learn.microsoft.com/en-us/power-platform/admin/admin-documentation

→Security & governance documentation

The hub for access control, DLP policies, and tenant-level governance across the platform.

learn.microsoft.com/en-us/power-platform/admin/security

→Security in Microsoft Dataverse (white paper)

The deep dive on the data-layer security model — business units, security roles, row/field-level security.

learn.microsoft.com/en-us/power-platform/admin/wp-security
3

Governance at scale

The core of the admin mandate — moving from an open default environment to a governed platform.

→Adoption Methodology

Microsoft's framework for admin, nurture, and governance strategy — the "how do I stand this up properly" playbook for scaling safely.

learn.microsoft.com/en-us/power-platform/guidance/adoption/methodology

→Power Platform white papers

In-depth guidance on governance, licensing, and security — the reference shelf for tricky decisions.

learn.microsoft.com/en-us/power-platform/guidance/white-papers/
4

Building it right

Solid delivery practices so what gets built is reliable and maintainable.

→Application Lifecycle Management (ALM)

Solutions, environment strategy, and pipelines to promote work Dev → Test → Prod in a controlled, repeatable way.

learn.microsoft.com/en-us/power-platform/alm/

→Power Platform Well-Architected

Pillar-based guidance (reliability, security, operational excellence, performance, experience) for designing well-governed solutions.

learn.microsoft.com/en-us/power-platform/well-architected/
A note on the CoE Starter Kit: you may see the “Center of Excellence (CoE) Starter Kit” referenced in older blogs and videos. Microsoft is transitioning its capabilities into the Power Platform admin center, so it’s intentionally left off this list — build your governance on the admin center and the Adoption Methodology above instead.
